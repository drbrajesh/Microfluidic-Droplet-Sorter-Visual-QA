# Dataset Description

## Overview

This dataset contains rendered inspection images from a synthetic microfluidic droplet-sorting chip. Each scene shows a Y-junction chip with colored droplets, a ringed target droplet, a counting chamber, electrode gates, possible clogs, bubbles, satellite droplets, and scanner-like visual artifacts.

The raw data is generated by the included deterministic script `generate_raw.py`. The generator records a hidden scene trace, then creates multiple-choice visual questions whose answers are derived from that trace. This makes labels objective while requiring solvers to inspect the image rather than infer from text alone.

The challenge is designed as visual question answering, not ordinary image classification. Each image can support multiple different questions, and the private split emphasizes hard visibility and OOD rendering cases.

## File Structure

| File or Folder | Rows / Files | Description |
|---|---:|---|
| `data.csv` | 2,700 rows | One row per question. Contains raw IDs, image path, question text, answer choices, answer label, hidden evaluation groups, and trace columns removed during preparation. |
| `images/` | 900 PNG files | Rendered 160x160 RGB chip inspection images. |
| `generate_raw.py` | 1 file | Deterministic dataset generator with fixed random seed. |

The prepare script remaps raw scene and question IDs to opaque IDs, copies every image to the public folder with a hashed filename, performs a scene-level train/test split, removes hidden trace columns, and creates `train.csv`, `test.csv`, `sample_submission.csv`, and private `answers.csv`.

## Raw Columns

| Column | Type | Description |
|---|---:|---|
| `question_id` | string | Raw question identifier. Remapped during preparation. |
| `scene_id` | string | Raw scene identifier. Remapped during preparation. |
| `image_path` | string | Relative path to the scene PNG. |
| `question_type` | string | One of six visual reasoning types. |
| `question` | string | Natural-language multiple-choice question. |
| `choice_a` to `choice_d` | string | Answer choices. |
| `answer_label` | string | Correct answer: A, B, C, or D. |
| `difficulty` | string | Hidden group: easy, medium, or hard. |
| `visibility` | string | Hidden visual condition: clear, noisy, low_contrast, or crowded. |
| `layout_family` | string | Hidden chip layout family. |
| `ood_axis` | string | Hidden rendering shift such as partial occlusion, dim fluorescence, or dense debris. |
| `trace_*` | mixed | Generator trace columns used to derive answers. Removed from public files. |

## Question Types

| Type | What It Tests |
|---|---|
| `target_outlet` | Locate the ringed target droplet among four channel regions. |
| `clog_location` | Identify where a dark obstruction appears. |
| `fluorescent_count` | Count bright green droplets in the counting chamber. |
| `gate_state` | Identify the active red electrode pair. |
| `bubble_position` | Locate a bright bubble or determine none is present. |
| `satellite_side` | Determine whether small satellite droplets appear upstream, downstream, both, or not visible. |
